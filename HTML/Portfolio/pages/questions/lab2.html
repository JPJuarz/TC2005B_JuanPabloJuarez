Lab 2 CSS

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Mi recomendación es evitar el uso de !important en CSS, ya que puede dificultar el mantenimiento del código y crear conflictos de estilos.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Se debe escoger con cuidado porque una imagen de fondo puede afectar al usuario en cuanto a la legibilidad de la pagina junto con afectar el rendimiento afectando toda la experiencia del usuario.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Como ingeniero de software, recomiendo usar porcentajes para diseños responsivos, píxeles para precisión visual en elementos específicos y evitar puntos ya que estos están orientados a impresión y pueden causar inconsistencias en distintos dispositivos y tamaños de pantalla.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

El CSS minimizado reduce el tamaño del archivo al eliminar espacios, comentarios y caracteres innecesarios lo cual resulta en un tiempo de descarga y procesamiento menor del navegador, mejorando la velocidad de carga del sitio y a la vez la experiencia del usuario.
